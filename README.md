# Dental Clinic — Landing & Booking Page

A modern, animated landing page for a dental clinic — hero, services, doctors, testimonials, and an appointment booking section.

**Stack:** Next.js 16 (App Router) · React 19 · Tailwind CSS 4 · Framer Motion

## Sections

- `Hero` — animated intro
- `About` — clinic overview
- `Services` — treatment list
- `Doctors` — staff profiles
- `Testimonials` — patient reviews
- `MakeAppointment` — booking CTA
- `components/motion/` — reusable scroll/reveal/stagger animation wrappers built on Framer Motion

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
